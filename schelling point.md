# Schelling Point

- A [[focal point]] in [[game theory]].
  - [[pull]] [[focal point]]
  - Named after [[thomas schelling]]
  - This [[agora]] tries to be a [[schelling point]]
    - It is an index of sorts. That is, it contains in itself a list of [[schelling points]]. You can push to [[schelling points]] or write there to add items to the list.
