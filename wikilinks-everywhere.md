# Wikilinks Everywhere

- A [[project]]. [[open source]].
- [[goal]] to make [[wikilinks]] work everywhere on the internet.
  - [[wikilinks]] can be [[lazily resolved]] in a user chosen [[context]].
    - Default context: an [[Agora]].
- Has a [[browser extension]] component.
  - The [[user]] can install an extension and see "dead" [[wikilinks]] "come to life" anywhere around the internet.
  - This makes it so that people can start using [[wikilinks]] on any platform that supports user generated [[content]] -- like any [[social network]].
